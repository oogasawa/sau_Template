## navbar自体の編集

元はこのような内容

```
    navbar: {
      title: 'Research00',
      logo: {
        alt: 'My Site Logo',
        src: 'img/logo.svg',
      },
      items: [
        {
          type: 'doc',
          docId: 'intro',
          position: 'left',
          label: 'Tutorial',
        },
        {to: '/blog', label: 'Blog', position: 'left'},
      ],

```

これを例えば以下のように直せば良い。

```
    navbar: {
        title: 'NIG Supercomputer',
        logo: {
            alt: 'NIGSC Logo',
            src: 'img/nigsc_logo.png',
        },
        items: [
            {
                to: "guides/overview",
                docId: 'overview',
                position: 'left',
                label: '利用案内',
            },
            {
                to: 'application/application',
                docId: 'application',
                position: 'left',
                label: '利用申請等',
            },
            // ...
            {
                type: 'localeDropdown',
                position: 'right',
            }

        ],
    },
```
